# Food Delivery Analytics Dashboard (Power BI)

##  Project Overview
This project is an interactive Power BI dashboard developed to analyze food delivery performance metrics such as revenue, delivery time, and customer ratings.

## 📊 Key Metrics
- Total Order Value: 4K
- Total Orders: 10
- Average Delivery Time: 35.20 mins
- Average Rating: 4.13

## Features
- City-wise order value analysis
- Restaurant performance comparison
- Delivery time tracking by Order ID
- Dynamic filtering using slicers

## Tools Used
- Power BI
- DAX
- Data Modeling
- Excel
